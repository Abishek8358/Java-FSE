# JavaScript Exercises: Local Community Event Portal

This directory contains the solutions to all 6 modules in the JavaScript (JS) exercise series. Each solution is built inside a separate HTML page paired with its own custom, external `.js` script file. The codebase implements the theme of a **Local Community Event Portal** and demonstrates dynamic, interactive client-side operations.

## 📂 File Directory Structure

*   📄 **`index.html`** - Master interactive JS dashboard listing and linking all exercises.
*   📄 **`ex_1.html` & `main.js`** - Exercise 1: Script setup, loading notifications (`alert`), and console logging (`console.log`).
*   📄 **`ex_2.html` & `main_ex2.js`** - Exercise 2: Syntax data types, re-assignable counters, template literals, and seat availability increment/decrement.
*   📄 **`ex_3.html` & `main_ex3.js`** - Exercise 3: User stories filtered via conditionals (`if-else`), rendering lists using `forEach()`, and wrapping registrations in `try-catch` blocks.
*   📄 **`ex_4.html` & `main_ex4.js`** - Exercise 4: Dynamic addition functions, category registration closure trackers, and callback arguments passed to higher-order filters.
*   📄 **`ex_5.html` & `main_ex5.js`** - Exercise 5: Constructor blueprints (`CommunityEvent`), prototype availability checks, and property listings via `Object.entries()`.
*   📄 **`ex_6.html` & `main_ex6.js`** - Exercise 6: Array mutations (`.push()`), music category isolation (`.filter()`), and card formatting helpers (`.map()`).

## 🚀 How to Run the Exercises

1. Navigate to the `js/` folder.
2. Double-click **`index.html`** to open the main exercise dashboard in any web browser.
3. Click on any card to launch that specific module.
4. Interact with the buttons, selectors, and text fields on the pages. If you want to inspect details, open the browser developer console by pressing `F12` or `Ctrl+Shift+I`.
