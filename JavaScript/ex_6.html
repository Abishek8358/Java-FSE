Array Methods Used:

  • .push(): Inserts the user's input object directly into the end of the array.
  • .filter(): Returns a subset of the original array (matching only category "Music"), leaving the original untouched.
  • .map(): Transforms each event object into a string block of HTML elements (prefixing names with "Workshop on " or "Session of " depending on category rules).
