# CSS3 Exercises: Local Community Event Portal

This directory contains the solutions to all 7 modules in the CSS3 exercise series. Each solution is built inside a separate HTML file paired with its own custom CSS stylesheet. The codebase implements the theme of a **Local Community Event Portal** and is styled entirely with pure CSS3 (no external styling libraries or Bootstrap).

## 📂 File Directory Structure

*   📁 **`images/`** - Contains assets like the generated tileable geometric body background pattern (`background.png`).
*   📄 **`index.html`** - Master pure CSS3 dashboard with grid layout and cards to launch each exercise.
*   📄 **`ex_1.html` & `styles_ex1.css`** - Exercise 1: Applying inline headings, embedded body tags, and linking external sheets with comments.
*   📄 **`ex_2.html` & `styles_ex2.css`** - Exercise 2: Maintaining formatted rules, clean nesting indentation, and team style guide comments.
*   📄 **`ex_3.html` & `styles_ex3.css`** - Exercise 3: Playground mapping universal (`*`), type (element), unique ID, class, and grouping selectors.
*   📄 **`ex_4.html` & `styles_ex4.css`** - Exercise 4: Setting HEX colors, transparent RGBA overlays, background fallbacks, and linear gradients.
*   📄 **`ex_5.html` & `styles_ex5.css`** - Exercise 5: Importing Google webfonts (`Space Grotesk`) and applying text-align, text-transform, letter-spacing, and line-height.
*   📄 **`ex_6.html` & `styles_ex6.css`** - Exercise 6: Link state customization (:link, :hover, :visited, :active) and horizontal list navigation menus.
*   📄 **`ex_7.html` & `styles_ex7.css`** - Exercise 7: Collapsed border admin tables, center cell alignment, and zebra striping via `nth-child(even)` rules.

## 🚀 How to Run the Exercises

1. Navigate to the `css/` folder.
2. Double-click **`index.html`** to open the main exercise dashboard in any web browser.
3. Hover over the cards and click **"Launch Module"** to view and interact with individual styling tasks.
