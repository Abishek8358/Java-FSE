Exercise 1: Applying Styles

This page demonstrates the implementation of the three main methods of incorporating CSS into HTML: Inline, Internal (embedded), and External.

Methods Implemented:

  • Inline Style: Applied directly to the main page heading <h1 style="color: #f43f5e;"> to force a custom pink/red color.
  • Internal Style: Declared inside the <style> element in the document head, setting the body background (#0f172a) and header layout.
  • External Style: Linked via styles_ex1.css, which handles modular, reusable layouts like the card container class (.info-card) and typography alignment.
