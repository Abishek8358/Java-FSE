Why Documentation Comments Matter:

CSS comments (defined using /* comment content */) are ignored by browsers but help organize stylesheets. They are used to create headers, segment code, and describe tricky hacks: 

/* Style for main CTA button */
.cta-button {
    background-color: #2563eb; /* Corporate royal blue */
    color: #ffffff;
    font-weight: 600;
}
