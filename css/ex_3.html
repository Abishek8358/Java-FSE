Selector Legend:

  • * (Universal): Resets all margins and paddings globally to 0.
  • h2 (Element/Type): Styles all subheadings uniformly.
  • #mainHeader (ID): Uniquely styles the banner dashboard container.
  • .eventCard (Class): Defines layout and background padding rules for each event card.
  • h3, p (Grouping): Simultaneously applies font family configurations to both selectors in a single block.
