Table Properties Implemented:

  • border-collapse: collapse; Merges cell borders together, eliminating default double-borders.
  • text-align: center; Centers the content inside headers (th) and data elements (td) identically.
  • padding: 14px 20px; Creates comfortable grid breathing room inside table blocks.
  • tr:nth-child(even) Selects every second row dynamically to apply alternating background coloring (Zebra Striping).
